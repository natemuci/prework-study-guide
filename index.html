HTML

  • HTML stands for hypertext markup language
  • The "style" element defines the CSS styles associated with a website.
  • The "base" element define the base URL for a webpage
  • The "link" element connects an external resource to the HTML document
  • The "meta" element defines metadata such as the charater set, desripton, keywords,author, and viewport
  • The "script" element embeds client-side scripts in the HTML document
  • HTML is used to create the srtucture of a webpage
  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user

CSS

  • There are three ways to style a webpage using CSS: Inline CSS, Internal CSS , and External CSS
  • Inline CSS is used when a developer wants to style an element directly in the HTML file
  • The Internal Style Sheet hold rules fo the page in the "head" section of the HTML file
  • An External Style Sheet is a stand alone .css file linked from a web page, allowing the developer to apply rules to multiple web pages
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element

Git

  • Git is a version control system that lets you manage and keep track of each version of your project
  • Git commands always start with "git", and are used to save and track versions of a developers code
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and swithces to it

JavaScript

  • JavaScript is a programming language that allows users to interact with websites
  • Using JavaScript, developers can add effects, interactivity and much more to a website
  • An Operator is a mathematical symbol that produces a result based on tw values
  • Conditionals are code structures used to test if an expression is true or false
  • A Variable is a named container that allows us to store data in our code.
  • Control Flow is the order in which a computer executes code in a script.
